# How to Fix Your Portfolio Website

## Files You Need to Upload to GitHub:

1. **yasar-portfolio.html** → Rename to **index.html**
2. **profile.jpg** → Keep the name as **profile.jpg**

## Step-by-Step Instructions:

### Step 1: Download Both Files
- Download `yasar-portfolio.html`
- Download `profile.jpg`

### Step 2: Rename the HTML File
- On your computer, rename `yasar-portfolio.html` to `index.html`

### Step 3: Upload to GitHub
1. Go to your repository: https://github.com/yasarSad/yasar-sadozai-website
2. Delete the old `yasar-portfolio.html` file (click on it → click trash icon)
3. Click "Add file" → "Upload files"
4. Drag BOTH files:
   - `index.html` (the renamed file)
   - `profile.jpg`
5. Click "Commit changes"

### Step 4: Wait 1-2 Minutes
Your website will be live at: https://yasarsad.github.io/yasar-sadozai-website/
